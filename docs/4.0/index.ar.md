# وثائق Vapor

</div>

مرحباً بك في وثائق Vapor! إن Vapor هو إطار عمل ويب للغة Swift، يتيح لك كتابة الواجهات الخلفية وتطبيقات الويب وواجهات API وخوادم HTTP بلغة Swift. كُتب Vapor بلغة Swift، وهي لغة حديثة وقوية وآمنة توفر عدداً من المزايا مقارنةً بلغات الخوادم الأكثر تقليدية.

## البدء

إذا كانت هذه هي المرة الأولى التي تستخدم فيها Vapor، فتوجه إلى [التثبيت ← macOS](install/macos.md) لتثبيت Swift وVapor.

بمجرد تثبيت Vapor، اطّلع على [البدء ← مرحباً بالعالم](getting-started/hello-world.md) لإنشاء أول تطبيق Vapor خاص بك!
